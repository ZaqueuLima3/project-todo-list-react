# ReactJs project

## Command

#### Installing Project

```bash
# clone it
$ cd ~
$ mkdir www && cd www
$ git clone git@github.com:ZaqueuLima3/project-todo-list-react.git
```

Now, access your project diretory.
```bash
$ cd project-todo-list-react

# Install dependencies
$ yarn
```

#### Final Steps
start your project in `http://localhost:3000`
```bash
# start the project
$ yarn start
```
